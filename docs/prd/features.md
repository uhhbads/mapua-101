# Filters and Features

## Face Box and GPA
- Draw box per face; overlay random GPA text (for example, 1.0–4.0) refreshed per detection or on a timer; ensure legible font and contrast.

## Dog-Ear Lens
- Anchor ear sprites to eyebrow or temple landmarks; nose overlay to nose tip.
- Add slight bob or lag smoothing; basic depth ordering via landmarks; occlusion not required.

## Y2K Frame
- Border PNG.
- Downsample to roughly 360–480p, add noise, scanlines, chromatic aberration, LUT tint, then upscale; optional slight VHS roll.

## Custom Frame
- Border PNG composited last; transparent center.

## Filter Switching
- Simple cycle button or hotkey; default auto-rotate every N seconds is configurable.
